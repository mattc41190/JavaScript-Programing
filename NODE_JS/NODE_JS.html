Mac Installation

An object literal is the easiest way (though no way is particularly difficult) to create a JavaScript. With an object literal you both create and assign an object at once.

The example below display how to create an object in JavaScript using the object literal approach


var person = {
firstName:"John",
lastName:"Doe",
age:50,
eyeColor:"blue"
};
Using the object above in our javascript file we can access a particular HTML element via using the code below.

document.GetElementById('objectLiteralExample').innerHTML = person.firstName;

Windows Installation

Creating an object via this method accomplishes the same functionality as an object literal. There is no benefit to using the keyword "new" and it is not the suggested method, but is not deprecated and is completely functional


var animal = new Object();
animal.name = "Patches";
animal.humanYears = 15;
animal.animalYears = 50;
animal.coatColor = "brown";

Using the object above in our javascript file we can access a particular HTML element via using the code below.

document.GetElementById('newObjectExample').innerHTML = animal.name;
