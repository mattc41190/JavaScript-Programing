JavaScript

Introduction

This site explains how to go from basic knowledge of HTML and CSS to a moderate understanding of JavaScript, and the creation of clean production level code written in JavaScript.

In the future we hope to include sections on JavaScript development in the mobile sphere.

We assume you have made pages with HTML and CSS before and that you understand the basic elements of a website and what they do. HTML being the framework and content, CSS being the styling placed against that content and JavaScript making that content interactive.

This site will be taking advantage of HTML5 components and as such will require a basic understanding of HTTP and LocalHost. Do not worry if you don't know much about these things as that is what the first section goes over.
