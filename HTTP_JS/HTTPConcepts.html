HTTP Resources

HTTP stands for Hyper Text Transfer Protocol. It is the means by which resources on the internet are found and used. This unit covers the resources on the internet and how they are accessed via the HTTP protocol.

  1. HTTP Resources
  2. HTTP Messages

function car(make, model, year, transmission) {
this.make = make;
this.model = model;
this.year = year;
this.transmission = transmission;

this.modelYear = function()
{
console.log(this.model +", "+this.year);
}
}

var firstCar = new car("Toyota", "Truck X50", 1989, "Auto");
var currentCar = new car("Ford", "Focus", 2008, "Auto");
